# 알고리즘 & 자료구조 학습 로드맵 (완전판)

## 학습 목표
프로그래밍 기초 지식에서 시작하여 **모든** 주요 알고리즘과 자료구조를 교과 과정처럼 단계적으로 마스터합니다. 코딩테스트와 실무에 필요한 문제 해결 능력을 체계적으로 기릅니다.

## 학습 철학
- **빠짐없이**: 기초부터 고급까지 모든 핵심 개념 학습
- **순차적으로**: 각 단계가 다음 단계의 기초가 됨
- **체계적으로**: 이론 → 구현 → 문제 풀이 순서로
- **실용적으로**: 코딩테스트와 실무에 바로 적용 가능

## 학습 방법
- 각 주차는 독립적인 문서로 제공됩니다
- 이론 설명 → 코드 예제 → 문제 풀이 순서
- 각 문서는 `week-XXX.md` 형식으로 저장됩니다 (3자리)
- 실제 코드 작성과 문제 풀이를 통해 실력 향상
- 각 Phase마다 복습 및 실전 문제 풀이 주차 포함

## 전체 학습 계획 (55주 과정 = 약 1년 1개월)

> 💡 **학습 속도 조절**: 이해가 잘 안 되는 주차는 시간을 더 투자하세요. 천천히 확실하게!

1. **로드맵 확인**: 이 문서에서 현재 어느 단계인지 확인
2. **문서 요청**: "X주차 학습 문서를 만들어주세요" 라고 요청
3. **이론 학습**: 개념과 원리 이해하기
4. **코드 작성**: 직접 코드 구현해보기
5. **문제 풀이**: 제공된 문제들 풀어보기
6. **복습**: 이해 안 되면 다시 보기
7. **다음**: 완전히 이해한 후 다음 주차로

## 💡 학습 팁

### 효과적인 학습을 위해
✅ **직접 코딩**: 예제를 보기만 하지 말고 직접 작성
✅ **디버깅 연습**: 에러를 두려워하지 말고 고쳐보기
✅ **문제 풀이**: 매 주차마다 제공되는 문제 반드시 풀기
✅ **복습**: 주기적으로 이전 내용 되돌아보기
✅ **시간 재기**: 문제 풀 때 시간을 재면서 연습
✅ **블로그 작성**: 배운 내용을 정리하면 더 오래 기억

### 주의사항
❌ 코드만 복사하지 않기 (직접 작성)
❌ 이해 없이 암기하지 않기
❌ 문제 풀이 건너뛰지 않기
❌ 한 번에 여러 주차 진행하지 않기

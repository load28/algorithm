## 1. 프로그래밍이란?

### 1.1 프로그래밍의 정의

**프로그래밍(Programming)**은 컴퓨터에게 **"무엇을 어떻게 할지"** 명령을 내리는 것입니다.

마치 요리 레시피처럼:
```
레시피 (요리사에게 주는 명령)
1. 물 500ml를 끓인다
2. 라면을 넣는다
3. 3분 동안 끓인다
4. 스프를 넣는다
5. 완성!

프로그램 (컴퓨터에게 주는 명령)
1. 숫자 두 개를 입력받는다
2. 두 숫자를 더한다
3. 결과를 화면에 보여준다
4. 완성!
```

### 1.2 왜 프로그래밍을 배우나요?

✅ **자동화**: 반복적인 일을 컴퓨터가 대신하게 만들 수 있어요
✅ **문제 해결**: 복잡한 문제를 논리적으로 풀 수 있어요
✅ **창작**: 앱, 게임, 웹사이트 등을 만들 수 있어요
✅ **미래**: 모든 산업이 소프트웨어와 연결되고 있어요

### 1.3 프로그래머는 무엇을 하나요?

프로그래머는:
1. **문제를 이해**합니다
2. **해결 방법을 설계**합니다 (알고리즘)
3. **코드로 작성**합니다
4. **테스트**하고 **수정**합니다

## 2. 알고리즘이란?

### 2.1 알고리즘의 정의

**알고리즘(Algorithm)**은 **문제를 해결하는 단계적인 방법**입니다.

일상생활의 알고리즘:

**예제 1: 양치하기 알고리즘**
```
1. 칫솔을 꺼낸다
2. 치약을 칫솔에 짠다
3. 물을 적신다
4. 이를 닦는다 (2분)
5. 입을 헹군다
6. 칫솔을 씻는다
```

**예제 2: 가장 큰 수 찾기 알고리즘**
```
세 개의 숫자: 5, 12, 8

1. 첫 번째 숫자(5)를 "가장 큰 수"로 정한다
2. 두 번째 숫자(12)와 비교한다
   → 12가 더 크니까 12를 "가장 큰 수"로 바꾼다
3. 세 번째 숫자(8)와 비교한다
   → 12가 더 크니까 그대로 둔다
4. 결과: 12가 가장 큰 수!
```

### 2.2 좋은 알고리즘의 특징

✅ **정확성**: 올바른 결과를 내야 함
✅ **명확성**: 각 단계가 명확해야 함
✅ **입력과 출력**: 입력이 무엇이고 출력이 무엇인지 분명해야 함
✅ **유한성**: 언젠가는 끝나야 함
✅ **효율성**: 시간과 메모리를 적게 사용할수록 좋음

## 3. 프로그래밍 언어

### 3.1 프로그래밍 언어란?

사람은 한국어, 영어로 말하죠.
컴퓨터는 **프로그래밍 언어**로 "대화"합니다!

컴퓨터는 사실 **0과 1**만 이해해요 (기계어).
하지만 사람이 0과 1로 프로그래밍하기는 너무 어려워요.

그래서 사람이 이해하기 쉬운 **프로그래밍 언어**가 만들어졌습니다!

### 3.2 주요 프로그래밍 언어

**Python (파이썬)** 🐍
```python
print("안녕하세요!")
```
- 배우기 쉬움
- 읽기 쉬운 문법
- 데이터 분석, AI, 웹 개발 등에 사용
- **추천**: 초보자에게 최고!

**JavaScript (자바스크립트)** 💛
```javascript
console.log("안녕하세요!");
```
- 웹 브라우저에서 실행
- 웹사이트를 동적으로 만들 수 있음
- 프론트엔드 + 백엔드 모두 가능

**Java (자바)** ☕
```java
System.out.println("안녕하세요!");
```
- 안정적이고 빠름
- 안드로이드 앱 개발
- 대기업에서 많이 사용

**C++ (씨플플)** ⚡
```cpp
cout << "안녕하세요!" << endl;
```
- 매우 빠름
- 게임 엔진, 시스템 프로그래밍
- 배우기는 조금 어려움

**Go (고)** 🔵
```go
fmt.Println("안녕하세요!")
```
- 구글이 만든 언어
- 빠르고 간결함
- 서버 개발에 많이 사용

> 💡 **이 강의에서는**: 언어에 구애받지 않는 **알고리즘의 원리**를 배웁니다.
> 예제는 주로 **Python**으로 작성하지만, 원리를 이해하면 어떤 언어로도 구현할 수 있습니다!

## 4. 개발 환경 설정하기

프로그래밍을 하려면 **개발 환경**이 필요합니다.

### 4.1 필요한 것들

1. **프로그래밍 언어 설치** (예: Python)
2. **코드 에디터** (코드를 작성하는 프로그램)

### 4.2 Python 설치하기

**Windows:**
1. https://www.python.org/ 접속
2. "Download Python" 클릭
3. 설치 파일 실행
4. "Add Python to PATH" 체크박스 꼭 체크!
5. Install Now 클릭

**Mac:**
1. 터미널 열기
2. `brew install python3` 입력
   (Homebrew가 없다면 먼저 설치)

**확인:**
터미널 또는 명령 프롬프트에서:
```bash
python --version
```
또는
```bash
python3 --version
```

버전이 나오면 성공! (예: Python 3.11.0)

## 5. 첫 프로그램: "Hello World"

### 5.1 Hello World란?

프로그래밍을 배울 때 가장 먼저 만드는 프로그램입니다.
"나는 프로그래밍을 할 수 있다!"를 선언하는 의식 같은 거예요! 🎊

### 5.2 Python으로 Hello World

**방법 1: 파일로 실행**

1. 새 파일을 만들고 `hello.py`로 저장
2. 다음 코드 작성:

```python
print("Hello World!")
```

3. 터미널에서 실행:
```bash
python hello.py
```

**결과:**
```
Hello World!
```

**방법 2: 대화형 모드 (Python Interactive Shell)**

터미널에서 `python` 또는 `python3` 입력:
```python
>>> print("Hello World!")
Hello World!
```

### 5.3 코드 설명

```python
print("Hello World!")
```

- `print`: 화면에 출력하는 **함수**
- `()`: 함수에 전달할 내용을 담는 괄호
- `"Hello World!"`: 출력할 **문자열** (따옴표 안의 텍스트)

### 5.4 다양한 출력해보기

```python
print("안녕하세요!")
print("나는 프로그래머가 될 거야!")
print(2024)
print(10 + 20)
```

**결과:**
```
안녕하세요!
나는 프로그래머가 될 거야!
2024
30
```

## 6. 프로그래밍의 기본 구성 요소

앞으로 배울 내용 미리보기:

### 6.1 데이터 (Data)
프로그램이 다루는 정보
- 숫자: 10, 3.14, -5
- 문자열: "안녕", "Hello"
- 참/거짓: True, False

### 6.2 변수 (Variable)
데이터를 저장하는 상자
```python
나이 = 15
이름 = "철수"
```

### 6.3 연산 (Operation)
데이터를 처리하는 것
```python
10 + 5  # 더하기
10 - 5  # 빼기
10 * 5  # 곱하기
10 / 5  # 나누기
```

### 6.4 조건문 (Conditional)
상황에 따라 다르게 행동
```python
if 비가 온다:
    우산을 가져간다
else:
    우산을 놔둔다
```

### 6.5 반복문 (Loop)
같은 일을 여러 번
```python
10번 반복:
    "안녕하세요!" 출력
```

### 6.6 함수 (Function)
재사용 가능한 코드 묶음
```python
def 인사하기():
    print("안녕하세요!")
```

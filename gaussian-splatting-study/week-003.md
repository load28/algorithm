## 1. 피타고라스 정리 복습

### 1.1 피타고라스 정리란?

직각삼각형에서 세 변의 관계를 나타내는 정리입니다.

```
      |\
      | \  c (빗변)
    b |  \
      |   \
      |____\
        a

a² + b² = c²
```

**핵심:**
- a, b: 직각을 끼고 있는 두 변
- c: 빗변 (가장 긴 변)
- c = √(a² + b²)

### 1.2 예제로 이해하기

**예제 1:** a = 3, b = 4일 때 c는?
```
c² = 3² + 4²
c² = 9 + 16
c² = 25
c = √25 = 5
```

**예제 2:** a = 5, b = 12일 때 c는?
```
c² = 5² + 12²
c² = 25 + 144
c² = 169
c = √169 = 13
```

**예제 3:** a = 1, b = 1일 때 c는?
```
c² = 1² + 1²
c² = 1 + 1
c² = 2
c = √2 ≈ 1.414
```

💡 **유명한 피타고라스 수:**
- (3, 4, 5)
- (5, 12, 13)
- (8, 15, 17)
- (7, 24, 25)

## 2. 거리 공식 유도하기

### 2.1 거리 공식의 발견

2주차에서 배운 거리 공식을 이제 직접 유도해봅시다!

**문제:** 점 A(1, 2)와 점 B(4, 6) 사이의 거리는?

**단계별 유도:**

```
      B (4, 6)
      *
      |\
      | \
      |  \  d (거리)
      |   \
      |____\
      *     *
  A (1,2)   C (4,2)
```

**1단계:** 직각삼각형 만들기
- 점 C를 (4, 2)에 놓으면 직각삼각형 ABC 완성!

**2단계:** 가로 거리 계산
```
AC = |4 - 1| = 3
```

**3단계:** 세로 거리 계산
```
BC = |6 - 2| = 4
```

**4단계:** 피타고라스 정리 적용
```
AB² = AC² + BC²
AB² = 3² + 4²
AB² = 9 + 16
AB² = 25
AB = √25 = 5
```

### 2.2 일반 공식

두 점 P₁(x₁, y₁)과 P₂(x₂, y₂) 사이의 거리 d:

```
d = √[(x₂ - x₁)² + (y₂ - y₁)²]
```

**기억하는 방법:**
1. x 좌표의 차이를 구한다 → (x₂ - x₁)
2. y 좌표의 차이를 구한다 → (y₂ - y₁)
3. 각각 제곱해서 더한다 → (x₂ - x₁)² + (y₂ - y₁)²
4. 제곱근을 씌운다 → √[...]

### 2.3 다양한 예제

**예제 1:** A(0, 0)과 B(6, 8) 사이의 거리
```
d = √[(6-0)² + (8-0)²]
d = √[36 + 64]
d = √100
d = 10
```

**예제 2:** A(-1, 2)와 B(3, 5) 사이의 거리
```
d = √[(3-(-1))² + (5-2)²]
d = √[4² + 3²]
d = √[16 + 9]
d = √25
d = 5
```

**예제 3:** A(-2, -3)과 B(1, 1) 사이의 거리
```
d = √[(1-(-2))² + (1-(-3))²]
d = √[3² + 4²]
d = √[9 + 16]
d = √25
d = 5
```

**예제 4:** A(0, 0)과 B(1, 1) 사이의 거리
```
d = √[(1-0)² + (1-0)²]
d = √[1 + 1]
d = √2
d ≈ 1.414
```

## 3. 선분의 중점

### 3.1 중점이란?

선분의 **정확히 가운데 점**을 중점이라고 합니다.

```
A *--------M--------* B
         (중점)
```

**직관적 이해:**
- A에서 B로 가는 길의 절반 지점
- A와 B로부터 같은 거리
- AM = MB

### 3.2 중점 공식

두 점 A(x₁, y₁)과 B(x₂, y₂)의 중점 M은:

```
M = ((x₁ + x₂)/2, (y₁ + y₂)/2)
```

**의미:**
- x 좌표: 두 x 좌표의 평균
- y 좌표: 두 y 좌표의 평균

**왜 평균일까?**
- A에서 출발해서 B까지 가는 거리의 절반
- x는 (x₁에서 x₂로)의 중간
- y는 (y₁에서 y₂로)의 중간

### 3.3 중점 예제

**예제 1:** A(2, 4)와 B(6, 8)의 중점
```
M = ((2+6)/2, (4+8)/2)
M = (8/2, 12/2)
M = (4, 6)
```

**검증:**
- AM 거리: √[(4-2)² + (6-4)²] = √[4+4] = √8
- MB 거리: √[(6-4)² + (8-6)²] = √[4+4] = √8
- AM = MB ✓

**예제 2:** A(-1, 3)과 B(5, -1)의 중점
```
M = ((-1+5)/2, (3+(-1))/2)
M = (4/2, 2/2)
M = (2, 1)
```

**예제 3:** A(0, 0)과 B(10, 20)의 중점
```
M = ((0+10)/2, (0+20)/2)
M = (5, 10)
```

**예제 4:** A(-3, -5)와 B(3, 5)의 중점
```
M = ((-3+3)/2, (-5+5)/2)
M = (0, 0)
```
→ 원점이 중점!

### 3.4 역문제: 중점을 알 때

**문제:** 선분 AB의 중점이 M(3, 4)이고 A(1, 2)일 때, B는?

**풀이:**
```
중점 공식: M = ((x₁+x₂)/2, (y₁+y₂)/2)

x 좌표: 3 = (1+x₂)/2
        6 = 1 + x₂
        x₂ = 5

y 좌표: 4 = (2+y₂)/2
        8 = 2 + y₂
        y₂ = 6

답: B(5, 6)
```

## 4. 거리와 중점의 활용

### 4.1 삼각형의 둘레

세 점 A(0, 0), B(4, 0), C(2, 3)으로 이루어진 삼각형의 둘레는?

```
AB = √[(4-0)² + (0-0)²] = 4
BC = √[(2-4)² + (3-0)²] = √[4+9] = √13
CA = √[(0-2)² + (0-3)²] = √[4+9] = √13

둘레 = 4 + √13 + √13 = 4 + 2√13 ≈ 11.21
```

### 4.2 정삼각형 판별

세 점 A(0, 0), B(2, 0), C(1, √3)이 정삼각형을 이루는지 확인

```
AB = 2
BC = √[(1-2)² + (√3-0)²] = √[1+3] = 2
CA = √[(0-1)² + (0-√3)²] = √[1+3] = 2

AB = BC = CA = 2 → 정삼각형! ✓
```

### 4.3 원의 방정식

중심이 C(2, 3)이고 반지름이 5인 원 위의 점 P(x, y)는:

```
CP = 5
√[(x-2)² + (y-3)²] = 5
(x-2)² + (y-3)² = 25
```

이것이 **원의 방정식**입니다!

### 4.4 선분을 3등분하는 점

점 A(0, 0)과 B(6, 9)를 잇는 선분을 3등분하는 두 점은?

**1/3 지점:**
```
P₁ = (0 + (6-0)/3, 0 + (9-0)/3)
P₁ = (2, 3)
```

**2/3 지점:**
```
P₂ = (0 + 2(6-0)/3, 0 + 2(9-0)/3)
P₂ = (4, 6)
```

**일반 공식 (내분점):**
선분 AB를 m:n으로 내분하는 점:
```
P = ((nx₁ + mx₂)/(m+n), (ny₁ + my₂)/(m+n))
```

## 6. 3D 공간으로의 확장 (미리보기)

2D에서 배운 개념을 3D로 확장할 수 있습니다!

### 3D 거리 공식
두 점 P₁(x₁, y₁, z₁)과 P₂(x₂, y₂, z₂) 사이의 거리:

```
d = √[(x₂-x₁)² + (y₂-y₁)² + (z₂-z₁)²]
```

**예제:** A(1, 2, 3)과 B(4, 6, 15) 사이의 거리
```
d = √[(4-1)² + (6-2)² + (15-3)²]
d = √[9 + 16 + 144]
d = √169
d = 13
```

### 3D 중점 공식
```
M = ((x₁+x₂)/2, (y₁+y₂)/2, (z₁+z₂)/2)
```

💡 **패턴을 발견하셨나요?**
- 2D: x, y
- 3D: x, y, z
- 차원이 하나 더 추가될 뿐, 원리는 같습니다!

# 가우시안 스플래팅 학습 로드맵

## 학습 목표
중학교 1학년 수준의 수학 지식에서 시작하여 가우시안 스플래팅(Gaussian Splatting)을 이용한 3D 렌더링을 이해하는 데 필요한 수학적 기초를 단계적으로 학습합니다.

## 학습 방법
- 각 주차는 독립적인 문서로 제공됩니다
- 이전 주차의 내용을 기반으로 새로운 개념을 배웁니다
- 각 문서는 `week-XX.md` 형식으로 저장됩니다
- 예제와 그림을 통해 직관적으로 이해할 수 있도록 구성됩니다

## 전체 학습 계획 (16주 과정)

### Phase 1: 좌표와 벡터 (1-4주차)
**1주차: 좌표와 점**
- 2D 좌표계 이해하기
- 점의 위치 표현하기
- 거리 계산하기
- 실습: 좌표 평면에서 점 다루기

**2주차: 벡터의 기초**
- 벡터란 무엇인가?
- 벡터의 크기와 방향
- 벡터의 덧셈과 뺄셈
- 실습: 벡터로 이동 표현하기

**3주차: 3D 좌표와 공간**
- 3D 좌표계 이해하기
- 3차원 공간에서의 점
- 3D 거리 계산
- 실습: 3D 공간에서 점 다루기

**4주차: 벡터 연산**
- 스칼라곱 (벡터의 크기 조절)
- 내적(Dot Product)
- 외적(Cross Product) 기초
- 실습: 벡터 연산 응용

### Phase 2: 행렬과 변환 (5-8주차)
**5주차: 행렬의 기초**
- 행렬이란 무엇인가?
- 행렬의 표기법
- 행렬의 덧셈과 뺄셈
- 실습: 간단한 행렬 계산

**6주차: 행렬 연산**
- 행렬의 곱셈
- 전치 행렬
- 단위 행렬과 역행렬
- 실습: 행렬 곱셈 연습

**7주차: 선형 변환**
- 변환이란?
- 이동(Translation)
- 크기 조절(Scaling)
- 실습: 2D 도형 변환하기

**8주차: 회전 변환**
- 2D 회전
- 3D 회전 기초
- 회전 행렬
- 실습: 점과 도형 회전시키기

### Phase 3: 함수와 가우시안 (9-12주차)
**9주차: 함수와 그래프**
- 함수란 무엇인가?
- 함수의 그래프
- 지수 함수 기초
- 실습: 다양한 함수 그려보기

**10주차: 가우시안 함수 기초**
- 종 모양 곡선
- 평균과 분산
- 1D 가우시안 함수
- 실습: 가우시안 곡선 그리기

**11주차: 2D 가우시안**
- 2차원 평면에서의 가우시안
- 원형 가우시안
- 타원형 가우시안
- 실습: 2D 가우시안 시각화

**12주차: 3D 가우시안**
- 3차원 공간에서의 가우시안
- 가우시안의 모양과 방향
- 공간에서의 분포
- 실습: 3D 가우시안 이해하기

### Phase 4: 가우시안 스플래팅 (13-16주차)
**13주차: 공분산 행렬**
- 공분산이란?
- 공분산 행렬의 의미
- 가우시안과 공분산의 관계
- 실습: 공분산으로 가우시안 조절하기

**14주차: 가우시안 스플래팅 개념**
- 스플래팅이란?
- 점을 가우시안으로 표현하기
- 여러 가우시안의 합성
- 실습: 간단한 스플래팅 예제

**15주차: 3D에서 2D로 투영**
- 3D 공간의 가우시안
- 카메라와 투영
- 투영된 가우시안
- 실습: 투영 변환 이해하기

**16주차: 가우시안 스플래팅 렌더링**
- 렌더링 파이프라인
- 여러 가우시안 합치기
- 색상과 투명도
- 최종 정리 및 응용

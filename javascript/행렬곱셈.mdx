#### 문제

2차원 행렬 arr1과 arr2를 입력받아, arr1에 arr2를 곱한 결과를 반환하는 함수, solution을 완성해주세요.

#### 제약조건

- 행렬 arr1, arr2의 행과 열의 길이는 2 이상 100 이하입니다.
- 행렬 arr1, arr2의 원소는 -10 이상 20 이하인 자연수입니다.
- 곱할 수 있는 배열만 주어집니다.

#### 입출력 예시

| arr1                              | arr2                              | return                                     |
| --------------------------------- | --------------------------------- | ------------------------------------------ |
| [[1, 4], [3, 2], [4, 1]]          | [[3, 3], [3, 3]]                  | [[15, 15], [15, 15], [15, 15]]             |
| [[2, 3, 2], [4, 2, 4], [3, 1, 4]] | [[5, 4, 3], [2, 4, 1], [3, 1, 1]] | [[22, 22, 11], [36, 28, 18], [29, 20, 14]] |

#### 풀이 과정

1. 행렬의 곱은 한쪽의 행과 다른쪽의 열이 같은때 연산이 가능함.
2. 이에 따라 최종적으로 만들어지는 행렬 배열을 만듬
3. 최종적으로 만들어지는 행렬의 크기만큼 순회함
4. 한쪽의 열을 순회함과 동시에 다른 쪽은 동일한 열을 가진 행을 순회하도록 함
5. 이에 따라 계산을 누적하여 더함

#### 정답

- 행렬의 곱은 m _ n 행렬과 n _ p 행렬의 곱은 m \* p 행렬이 됨을 알아야함
- 결과 크기를 만들어 놓고 내적 계산 로직을 수행함

```javascript
function solution(arr1, arr2) {
  const answer = [];
  const row1 = arr1.length;
  const col1 = arr1[0].length;
  const row2 = arr2.length;
  const col2 = arr2[0].length;

  for (let i = 0; i < row1; i++) {
    answer.push(new Array(col2).fill(0));
  }

  for (let i = 0; i < row1; i++) {
    for (let j = 0; j < col2; j++) {
      for (let k = 0; k < col1; k++) {
        answer[i][j] += arr1[i][k] * arr2[k][j];
      }
    }
  }

  return answer;
}
```
